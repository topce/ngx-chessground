aiPlay(cg: Api, chess: ChessInstance, delay: number, firstMove: boolean)

Executes an AI move in a chess game after a specified delay.

Parameters :
Name Type Optional Description
cg Api No
  • The chessground API instance.
chess ChessInstance No
  • The chess.js instance.
delay number No
  • The delay in milliseconds before the AI makes a move.
firstMove boolean No
  • A boolean indicating if this is the first move of the game.

toColor
toColor(chess: ChessInstance)

Converts the current turn of a chess game to a color string.

Parameters :
Name Type Optional Description
chess ChessInstance No
  • An instance of a chess game.
Returns : Color

The color string "white" if it's white's turn, otherwise "black".
toDests
toDests(chess: ChessInstance)

Generates a map of possible destination squares for each piece on the board.

Parameters :
Name Type Optional Description
chess ChessInstance No
  • An instance of the Chess game.
Returns : Map<Key, Key[]>

A map where the keys are the squares with pieces that have legal moves, and the values are arrays of destination squares for those pieces.
